# Member 1 Contribution Summary

As **Member 1: Data Engineering & Preprocessing Lead**, I designed and implemented the dataset pipeline for **Brainrot to English: Agent-Driven Diachronic Linguistic Analysis**.

My contribution includes:

- I designed the dataset folder structure and processing workflow.
- I prepared slang seed terms and a structured slang dictionary.
- I implemented Urban Dictionary-style data ingestion with retry handling and a curated fallback.
- I implemented local Twitch corpus ingestion without direct platform scraping.
- I implemented source merging while preserving metadata.
- I implemented text preprocessing, slang detection, unknown term detection, sentiment labelling, confidence scoring, and quality flagging.
- I generated train, validation, and test datasets for later model fine-tuning.
- I generated data quality reports and dataset statistics.
- I documented source strategy, preprocessing methodology, dataset schema, ethical considerations, and data limitations.

This work prepares a clean, transparent, and extensible dataset foundation for the later modelling, API, database, and active learning components of the group project.
